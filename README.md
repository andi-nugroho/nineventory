# NINEVENTORY - Smart inventory management with AI-powered insights.

Nineventory is a comprehensive inventory management system designed to streamline the process of tracking assets, managing loan requests, and facilitating administrative approvals. It provides a modern, user-friendly interface for both administrators and standard users to interact with company assets efficiently.

![Preview Website](preview-website.png)

## Features

- **Dashboard Overview**: Provides real-time statistics on total inventory, available stock, active loans, and pending approvals.
- **Inventory Management**: Complete catalog browsing with search and filtering capabilities.
- **Loan System**: Streamlined process for users to request equipment and for administrators to manage those requests.
- **Approval Workflow**: dedicated section for administrators to approve or reject loan requests.
- **User History**: Users can track their previous loans and current request status.
- **AI Chatbot**: Integrated AI assistant to help users check stock availability and answer policy questions.

## Tech Stack

### Frontend
- **Framework**: HTML5, Vanilla PHP
- **Styling**: [Tailwind CSS](https://tailwindcss.com) (via CDN for rapid UI development)
- **Interactivity**: [Alpine.js](https://alpinejs.dev) (for lightweight reactive components)
- **Icons**: [Lucide Icons](https://lucide.dev)
- **Fonts**: [DM Sans](https://fonts.google.com/specimen/DM+Sans) (Google Fonts)

### Backend
- **Language**: PHP 8.2+
- **Database**: MySQL / MariaDB
- **Database Interaction**: PDO (PHP Data Objects) for secure database queries
- **Architecture**: MVC-inspired modular structure

### Tools & Environment
- **Server**: Apache (via XAMPP)
- **Version Control**: Git & GitHub
- **Editor**: VS Code
